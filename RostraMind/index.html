Core Features & Capabilities

Flexible Grid Configuration

Specify custom date rows, header rows, header columns, and employee email columns. RostraMind adapts to your spreadsheet's structure dynamically.

Multi-Slot Custom Scheduling

Set up custom shift IDs (like Forenoon "FN", Afternoon "AN") with default durations. Map spreadsheet cell values to exact calendar event durations.

Day-of-Week Overrides

Need custom shift hours for Friday afternoons or weekend rosters? Configure day-of-week overrides to dynamically modify slot timings.

Auto Reminders & Push Logs

Set custom event titles, descriptions, and multiple Google Calendar reminders (popups or emails) scheduled X minutes before the event begins.

Custom Calendar ID Support

Target team members' primary calendars or designate a specific calendar resource ID for team schedules or shared calendars.

Real-Time Sync Logs

Run sync processes directly from the sidebar. Inspect a scrollable, detailed execution log mapping additions, skips, and calendar updates.
