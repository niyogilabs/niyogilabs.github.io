RostraMind Support & Setup

Learn how to configure your spreadsheet grids, map custom shift slots, add reminders, and troubleshoot sync behaviors.

Setup & Configuration Manual

Follow these steps to map your Google Sheet roster and synchronize it directly with Google Calendar.

1

Open configuration Sidebar

From your active Google Spreadsheet, navigate to the menu: Extensions → RostraMind → Configure. This opens the control panel on the right side of your spreadsheet.

2

Map the Grid Configuration

In the Setup tab, map the layout of your sheet roster using the 1-based indexing values:

  • Target Sheet Name: Select the sheet tab where your roster lives.
  • Header Rows: Number of metadata/header rows at the top of the spreadsheet (default is 5).
  • Header Cols: Number of fixed column headers on the left (default is 2).
  • Date Row: The row containing the shift dates (default is row 2).
  • Slot Row: The row containing shift codes like FN/AN (default is row 4).
  • Email Column: Column containing staff email addresses (default is column 1).
3

Configure Shift Slots & Overrides

Go to the Manage Slots tab to define default hours for shift codes:

  • Example: Code FN (Forenoon) runs from 09:15 to 12:30.
  • Example: Code AN (Afternoon) runs from 13:15 to 16:30.
  • Day Overrides: Expand a slot and add customized hours for specific days (e.g. override AN on Fridays to run from 13:45 to 17:00).
4

Add Notifications & Save

In the Calendar tab, customize the scheduled event title, description, and add automatic calendar notifications (e.g. Email or Push notification 30 minutes before shifts). Press Save All Settings.

5

Synchronize Calendars

To trigger synchronization, navigate to the Google Spreadsheet menu: Extensions → RostraMind → Sync Calendars. An execution log will display calendar additions, updates, and skips.
