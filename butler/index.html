Local-First IoT Platform

Meet the Butler IoT Platform

A robust, offline-first IoT ecosystem built to automate physical activities. Designed to operate reliably through power drops and intermittent network coverage, delegating tasks to technology to simplify human lives.

Request IoT Customization Butler (Device Firmware) ButlerManager (Android App)

Two Parts, One IoT System

1. Butler Device Firmware

The core hardware controller code written in C++ that resides directly on physical microcontrollers. Built to automate physical activities locally, it operates with absolute independence from cloud connections.

  • Automates physical switches, relays, and motor actions
  • Robust failure handling during power interruptions
  • Maintains system state offline in non-volatile flash
  • Lightweight C++ code compile-optimized for IoT chips
View Firmware Code

2. ButlerManager Android App

The Android-based companion dashboard. It acts as the local administrative interface to configure parameters, map hardware pins, monitor offline telemetry, and manage local alerts.

  • Local network auto-discovery of Butler controllers
  • Real-time physical telemetry reporting (power, metrics)
  • Manual override console and scheduling calendars
  • Companion Android application for offline configuration
View Android App Code

Robust Physical Automation

We believe hardware automation should never depend on external clouds. Get your physical activities automated with custom IoT hardware setups.

Local State Preservation

We program custom recovery routines into the device firmware, ensuring that your machinery safely resumes its scheduled physical automation activities instantly upon power restoration.

Custom UI Layouts

Get customized Android views inside your ButlerManager app that display the exact buttons, dial indicators, and status indicators needed for your physical operations.

Intermittent Network Sync

We build secure store-and-forward local databases that gather physical logs offline, queuing sync events until an active network handshake is available.
